# HR Resume Matcher

Консольная утилита для оценки соответствия резюме вакансии через локальный Claude CLI.

Помогает кандидату:
- Понять, подходит ли он на позицию (оценка 0-100)
- Узнать пробелы и риски
- Получить рекомендацию по зарплате (USD и RUB)
- Получить готовый ответ для HR-рекрутера
- Подготовиться к собеседованию

## Требования

- Python 3.11+
- [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) в PATH

## Установка

```bash
python3 -m venv .venv
.venv/bin/pip install PyPDF2 python-docx
```

## Использование

```bash
# Вакансия текстом
.venv/bin/python hr_match.py resume.pdf --vacancy "Senior DevOps Engineer, remote..."

# Вакансия из PDF
.venv/bin/python hr_match.py resume.pdf --vacancy job.pdf

# Своё резюме + вакансия
.venv/bin/python hr_match.py resume.pdf --vacancy job.pdf

# Вакансия по URL
.venv/bin/python hr_match.py resume.pdf --vacancy https://example.com/job.pdf

# Сохранить отчёт
.venv/bin/python hr_match.py resume.pdf --vacancy job.pdf --out report.txt
```

Без аргументов покажет подробную справку:

```bash
.venv/bin/python hr_match.py
```

## Как это работает

Программа вызывает `claude --print` через subprocess — никаких API-ключей и HTTP-запросов. Весь анализ делает локально установленный Claude Code.
